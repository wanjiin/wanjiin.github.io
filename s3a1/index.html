Monsters Breaking Penn Face
“Penn Face” is the term that Penn students use to describe what they often see on campus: peers who seem to be projecting an air of invincibility, of calm, as if they have everything figured out.

But this is rarely ever the true case.

To emphasize this dichotomy, we initially created characters that represent the way we physically present ourselves at Penn. Then, we used a script to create monsters that represent our more internal states. These characteristics were determined based on the following aspects of student life:
Assignments due this week Class color of most urgent Canvas assignment → skin color
10% increase in color saturation per assignment
Level of stress on a scale of (0 → 10) facial expression where 0 → happy / content and 10 → mental breakdown
Time you went to sleep last night bedtime hour → number of eyes
Time you woke up this morning (waking hour - 6) → number of arms
Missed lectures last week if none → no additions
if even → horns
if odd → wings
Caffeinated drinks this week color of your favorite drink → hair color
10% increase in color saturation per drink
Times you were anxious about the future this week hair length where 0 → bald and 10+ → floor length
